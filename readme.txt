== Installation ==

1. Upload the `hivefiliate` directory into the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. On the hivefiliate plugin just click settings or you can go to the WooCommerce > Hivefiliate section in your admin panel to add your Hivefiliate API keys and to enable the plugin.
